# 01 - إعداد وبداية المشروع (Project Setup)

## 1. إنشاء المشروع
تم إنشاء هذا المشروع باستخدام **.NET 9** كواجهة برمجية (Web API) باستخدام الأمر:
```bash
dotnet new webapi -n Clinic2026_API
```
هذا النمط يوفر بيئة حديثة وسريعة لبناء الـ RESTful APIs.

## 2. تنظيف المشروع الأولي (Cleanup)
بعد الإنشاء، قمنا بتنظيف الملفات الافتراضية مثل `WeatherForecast.cs` ووحدة التحكم `Controllers` الخاصة بها، لتهيئة المشروع ليكون نظيفاً وقابلاً للتوسع وفق احتياجات العيادة.

## 3. إعدادات الإطلاق (Launch Settings)
تم تعديل ملف `launchSettings.json` ليعمل التطبيق على منافذ محددة ويدعم التشغيل عبر `https` و `http`، مع تفعيل Swagger كواجهة للتجربة.
